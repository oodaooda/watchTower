# watchTower Documentation

This directory is the canonical documentation entry point for watchTower.

## Structure

- `architecture/`: system shape, engineering rules, and operating model
- `adr/`: architecture decision records
- `milestones/`: roadmap and phased delivery tracking
- `specs/`: feature and interface specifications

## Milestone Tags

- Milestones are tracked with IDs: `M###_slug`
- The roadmap is high-level only (ID + feature name + status)
- Detailed work items live in milestone checklist files

## Start Here

- Architecture overview: `architecture/system-overview.md`
- Engineering standards: `architecture/engineering-standards.md`
- Operations runbook: `architecture/operations-runbook.md`
- Active roadmap: `milestones/roadmap.md`
- Transcript spec: `specs/M001_earnings-call-transcripts.md`
