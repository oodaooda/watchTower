# G1 - Polymarket Taiwan

Last validated: 2026-05-17

## What It Measures

Implied probability from a pinned Taiwan-related Polymarket market. `POLYMARKET_TAIWAN_MARKET_ID` is required; the module fails closed instead of searching because search can select the wrong market.

## Source URL + Auth

- Source: Polymarket Gamma API
- URL: `https://gamma-api.polymarket.com/markets?id=<POLYMARKET_TAIWAN_MARKET_ID>`
- Auth: none

## Refresh Cadence

Every 5 minutes in the target design.

## Interpretation

Higher probability indicates rising geopolitical risk relevant to semiconductor and VGT-heavy exposure.

## Threshold Rules

- Green: `<= 10%`
- Amber: `> 10%`
- Red: `> 15%`

## Known Failure Modes

- Missing `POLYMARKET_TAIWAN_MARKET_ID` leaves the module unconfigured/grey.
- Market can close, resolve, or lose liquidity.
- Outcome labels and prices may be encoded as JSON strings or arrays.
