# Milestones

Use this folder for roadmap and phase-tracking documents.

## ID Convention

- Feature milestones use IDs and filenames in this format: `M###_slug`
- Example: `M001_earnings-call-transcripts-checklist.md`
- The roadmap tracks milestones at high level by ID and feature name only.

## Files

- `roadmap.md`: high-level milestone index
- `M001_earnings-call-transcripts-checklist.md`: detailed checklist for transcript feature rollout

When a phase becomes large enough to need its own closeout or retrospective, add a dedicated file here and link it from `roadmap.md`.
